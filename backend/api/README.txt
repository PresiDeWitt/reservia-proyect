README para la app 'api'

- models.py: Define los modelos de datos (tablas de la base de datos) como Restaurante, Reserva, Usuario, etc.
- views.py: Contiene la lógica de negocio y las vistas (endpoints) de la API.
- serializers.py: Convierte los modelos a formatos JSON y viceversa para la API REST.
- urls.py: Define las rutas/endpoints de la API.
- tests.py: Incluye pruebas unitarias y de integración para la app.
