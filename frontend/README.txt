README frontend

- public/: Archivos estáticos públicos (favicon, index.html, etc.).
- src/components/: Componentes reutilizables de React.
- src/pages/: Vistas principales de la aplicación (Home, Reservas, Perfil, etc.).
- src/services/: Lógica para llamadas a APIs y servicios externos.
- src/hooks/: Custom hooks de React.
- App.tsx, main.tsx: Entradas principales de la app.
- tailwind.config.js: Configuración de Tailwind CSS.
- package.json, tsconfig.json: Configuración de dependencias y TypeScript.
